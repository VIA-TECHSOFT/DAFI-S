DAFI'S - Mobilidade Urbana

  Este projeto consiste no desenvolvimento de um portal institucional e protótipo de aplicativo voltado para a DAFI'S, uma empresa do ramo de mobilidade urbana.

Sobre o Projeto

  O objetivo desta solução é transformar a experiência de transporte coletivo em Caieiras através de uma interface moderna, intuitiva e focada na experiência do usuário. O projeto integra os seguintes pilares de marca:

    Frota Moderna: Foco em qualidade e manutenção constante.

    Tecnologia: Monitoramento de rotas em tempo real.

    Conforto e Inclusão: Acessibilidade e facilidades digitais para todos os passageiros.

Funcionalidades do Aplicativo

  O protótipo do aplicativo DAFI'S foca em agilidade e controle para o passageiro, incluindo:

    Busca de Linhas: Consulta rápida de trajetos e destinos.

    Monitoramento em Tempo Real: Previsão de chegada dos ônibus.

    Gestão Financeira: Controle de saldo, histórico de recargas e extrato de uso do cartão de transporte.

Identidade Visual

  O design do projeto utiliza elementos de conexão e movimento, com uma paleta de cores institucional baseada em tons de azul (#03A6B6, #009DB0, #0076A8, #0D2B4D) que transmitem confiança e inovação.
